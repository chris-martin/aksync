org.codeswarm.aksync.TokenRetryInterval

ExponentialBackoff

case class ExponentialBackoff(min: FiniteDuration = ..., max: FiniteDuration = ..., base: Double = 1.2) extends TokenRetryInterval with Product with Serializable

min

The amount of time to wait after the first failure.

max

A maximum duration. The delay will grow up to this quantity and then remain at the maximum value indefinitely.

base

The base of the exponent used to calculate the delay's growth as the number of failures increases. Must be at least 1. If the base is exactly 1, then the delay will remain constant at the min value. If the base is 2, the delay doubles with each failure.
